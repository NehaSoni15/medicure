# 💊 Medicure - Online Medicine Store

A modern frontend web application for browsing and purchasing medicines online.

---

## 📖 Project Description

Medicure is a frontend web application designed to simulate an online medicine store. It provides a clean, responsive UI and core e-commerce features without using a backend.

Users can browse medicines, search products, add items to cart, manage quantities, and complete a checkout process. The application uses AngularJS for dynamic rendering and LocalStorage for storing user and cart data.

---

## 🚀 Features

* 🔐 User Authentication (Login / Register)
* 💊 Dynamic Medicine Listing (AngularJS + JSON)
* 🔍 Search Functionality
* 🛒 Add to Cart
* ➕ Increase / Decrease Quantity
* ❌ Remove Items from Cart
* 💰 Total Price Calculation
* ✅ Checkout Flow with Success Page
* 📱 Responsive UI using Bootstrap
* 🔔 Toast Notifications
